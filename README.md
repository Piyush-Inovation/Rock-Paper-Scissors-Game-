# Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. This project is beginner-friendly and focuses on logic building, DOM manipulation, and basic game mechanics.

---

##  Project Description

This Rock Paper Scissors game allows a user to play against the computer. The computer randomly selects its move, and the game determines the result (Win / Lose / Draw) based on standard rules.

The project is designed to practice:

* JavaScript logic and conditionals
* DOM manipulation
* Event handling
* Random number generation
